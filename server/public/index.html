How it works

Three steps to save a life

No Wi-Fi, no cell towers, no internet needed. Pure Bluetooth mesh — works anywhere on Earth.

01
📱
Install & Set Up
Download MeshAlert on Android. Enter your name and medical info in your profile — it gets broadcast with every SOS so first responders know who you are before they reach you.
02
📡
Mesh Connects Automatically
The app advertises via BLE and scans for nearby peers. Devices within 50m form a mesh automatically. Each device acts as a relay node, extending the network with every person nearby.
03
🆘
SOS Propagates the Mesh
Press SOS. Your alert — GPS, blood group, voice note — relays hop-by-hop through every nearby device until it reaches a gateway with internet and syncs to the command dashboard.

Transport
BLE 4.0+ GATT
Custom 128-bit UUID. Split advertisement + scan response to stay within the 31-byte BLE limit.
Deduplication
MessageID + Seen Set
Each message has a UUID. Devices track seen IDs to prevent relay loops. TTL decrements each hop.
Sync
Auto Cloud Upload
SyncService monitors connectivity. When internet returns, all queued SOS messages upload to Railway instantly.
Real-time
SSE Push Events
Dashboard uses Server-Sent Events — alerts appear on the map within milliseconds of arriving at the backend.
